# css-generator-with-nextjs

## Css Generator wit Next Js 

This app is basic css generator which is border, box-shadow, filter etc. 

**For node_modules folder**
```
npm install 
```

**Starting the app**
```
npm run dev 
```
**Bulding the app**
```
npm run build 
```

**For view**
https://css-generator-with-nextjs.vercel.app/
